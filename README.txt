1、images文件夹是整站的图片资源。
2、javascripts文件夹是整站的js文件。
3、jsons文件夹是个人中心-常见问题该功能的.json代码。
4、node_modules文件夹是前端开发人员在安装依赖包时自动生成，无需上传。
5、styles文件夹是整站CSS样式，各个样式文件中有说明该css样式的功能。
6、views文件夹是整站的HTML文件。
7、gulpfile.js文件为gulp配置文件
8、package.json文件为node配置文件，里面是前端开发中需要用到的依赖包信息

PS:
9.网站真正使用的css路径为:/h5/styles/jumi.css(该文件是所有css文件的汇总)
10.网站真正使用的js路径为:/h5/javascripts/dist(该文件包括功能js和页面js两个部分。都是压缩后的文件)
